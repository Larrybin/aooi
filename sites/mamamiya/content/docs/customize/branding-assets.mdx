## Default Asset Paths

- `public/logo.png`
- `public/favicon.ico`

## Runtime Updates

These file-based defaults live in `public/`. If admin uploads are enabled and storage is configured, admins can replace visible brand assets from the dashboard. The same assets feed docs, auth, not-found, and metadata surfaces.
