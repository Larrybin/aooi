## 默认素材路径

- `public/logo.png`
- `public/favicon.ico`

## 运行时更新

这些基于文件的默认素材放在 `public/` 里。如果已启用后台上传并配置了存储，管理员可以在仪表盘中替换可见的品牌素材。这些素材会被 docs、auth、not-found 和 metadata 页面复用。

## 引用位置

- docs 顶部导航
- auth 布局
- not-found 页面
- metadata 与预览图默认值
